# 🔌 TrendAura V2 - Hyper-Engine API Routing Protocols

يوضح هذا الدليل نقاط الاتصال والمسارات الخلفية (Endpoints) الحقيقية المعتمدة لتشغيل لوحة التحكم ومزامنة التوكنز والمدفوعات البنكية حياً في بيئة الإنتاج لعام **2026**.

---

## 🛡️ 1. قطاع الهوية والملفات الشخصية (`/api/auth`)
* **GET `/profile`**: جلب البيانات السيادية كاملة للمستخدم (الاسم، البريد، الرصيد، الباقة، الصورة) عند تحديث الصفحة لمنع تصفير العدادات.
* **PUT `/profile`**: تحديث وتطهير الاسم الكريم لصانع المحتوى في قاعدة البيانات.

## 🧠 2. قطاع معالجة الذكاء الاصطناعي ومحرك الفايرال (`/api/ai`)
* **POST `/generate`**: استقبال الفكرة السلوكية وصياغة السكريبت الفايرال المقسم إلى (Hook, Body, CTA) مع الهاشتاقات وحساب القيود (يستهلك 10 توكنز).
* **POST `/analyze-metrics`**: استجواب فحص احتمالية صعود المقطع وتقدير نسب ومنحنى بقاء واحتفاظ الجماهير (Retention Rate Curve).

## 💳 3. قطاع الفوترة وعمليات بوابة ميسر السعودية (`/api/payment`)
* **POST `/webhook`**: جدار استقبال إشارات السداد اللحظية والآمنة من بوابة ميسر (Moyasar) لشحن الرصيد التراكمي وتفعيل رتبة الباقة المدفوعة فوراً.

## 👑 4. قطاع التراخيص والحدود الفنية (`/api/subscription`)
* **GET `/details`**: الاستعلام الفوري عن قيود ومميزات باقة الاشتراك النشطة حالياً للمستخدم ومطابقتها بملف الثوابت.

## 📊 5. قطاع تتبع كوتا الاستهلاك اليومي (`/api/usage`)
* **GET `/daily`**: مراقبة حجم استهلاك المحاولات اليومية حياً لتغذية شريط الاستهلاك بالواجهة.
