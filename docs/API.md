# 🔌 TrendAura V2 - Hyper-Engine API Routing Protocols

يوضح هذا الدليل نقاط الاتصال والمسارات الخلفية (Endpoints) المعتمدة لتشغيل لوحة التحكم ومزامنة التوكنز والمدفوعات البنكية.

## 🔐 1. مسارات الهوية والتحقق (`/api/auth`)
* **POST `/register`**: إنشاء حساب جديد وحقن كود الإحالة إن وُجد (+50 توكن للمدعوين).
* **POST `/login`**: التحقق من الهوية وصرف توكن الجلسة السري المستقر في الحافظة المحلية.

## 🧠 2. مسارات التوليد والذكاء الاصطناعي (`/api/ai`)
* **POST `/generate-script`**: معالجة المدخلات وصياغة الخطاف الفايرال (يستهلك توكنز بناءً على رتبة الباقة).
* **POST `/analyze-retention`**: حصري لباقة **Viral Engine** فحص احتمالية صعود المقطع وتقدير نسب البقاء للتريند.

## 💳 3. مسارات الفوترة وبوابة ميسر (`/api/subscription`)
* **POST `/create-invoice`**: توليد رابط الدفع الآمن وتأمين الجلسة المالية مع Moyasar.
* **POST `/webhook`**: استقبال الإشارات اللحظية من البنك لتفعيل ميزات الباقة فوراً وتغذية جدول الفواتير.
* **POST `/claim-daily`**: معالجة مكافأة الامتصاص اليومية لشحن الحساب بـ 2 توكنز مجانية.
