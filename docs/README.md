## 🗂️ التقسيم الهيكلي لزاوية التحكم والملاحة (Sidebar Architecture)
تم هندسة لوحة التحكم بخمسة مسارات سيادية متكاملة ومربوطة بثوابت الـ `ROUTES` الصارمة:
1. **الرئيسية**: مقر مصفوفة التوليد الثلاثية الحية (Generator Box, Script Card, Metrics).
2. **السكربتات**: مستودع الأرشيف المتقدم وحساب الكلمات التراكمية ومعاينة السكربت الفعال.
3. **الاشتراكات**: شاشة كروت الأسعار النيونية الفخمة المتوهجة الحية للترقية.
4. **إدارة الاشتراك**: المركز المالي الموحد لرصد الباقة الحالية والتوكنز وسجل الفواتير التاريخي.
5. **الإعدادات**: تهيئة وتعديل البروفايل والمظهر وصياغة حواف الكروت (28px مقابل 12px).

## 🛠️ الترسانة البرمجية للمشروع (Tech Stack)
* **الواجهة الأمامية**: React.js + Vite (أداء فائق وحصانة ضد أخطاء الـ Syntax).
* **التنسيق البصري**: Tailwind CSS + ستايل Glassmorphic داكن نيوني متوهج ملوكي ناصع.
* **الباك إند والسيرفر**: Node.js + Express.js (دعم كامل لنظام الـ ES Modules).
* **قاعدة البيانات والهوية**: Supabase السحابي المطور لحماية وضبط الجلسات.
* **بوابة المدفوعات**: بوابة دفع ميسر (Moyasar) السعودية المؤمنة بالكامل بنسبة 100% لعمليات مدى وفيزا.
