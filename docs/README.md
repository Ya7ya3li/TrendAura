# 🌌 TrendAura V2 - المنصة الذكية الفاخرة لصنّاع المحتوى

منصة تكتيكية رائدة متكاملة (SaaS) مهندسة خصيصاً لصناعة وإدارة السكريبتات والأفكار الفيروسية (Viral Scripts) الموجهة لمنصات تيك توك وسوشيال ميديا، مدعومة بمحركات ذكاء اصطناعي تفحص نسب الاحتفاظ والـ Retention لتصدر الترندات العالمية.

## 🗂️ التقسيم الهيكلي لزاوية التحكم (Sidebar Architecture)
تم هندسة لوحة التحكم بخمس مسارات سيادية متكاملة:
1. **الرئيسية**: مقر مصفوفة التوليد الثلاثية الحية.
2. **السكربتات**: مستودع الأرشيف المتقدم وحساب الكلمات التراكمية ومعاينة السكربت الفعال.
3. **الاشتراكات**: شاشة كروت الأسعار النيونية الفخمة المتوهجة الحية.
4. **إدارة الاشتراك**: المركز المالي الموحد لرصد الباقة الحالية والتوكنز والفوترة اليومية.
5. **الإعدادات**: تهيئة البروفايل والمظهر + زر منطقة الخطورة لتطهير المسودات.

## 🛠️ الترسانة البرمجية للمشروع (Tech Stack)
* **الواجهة الأمامية**: React.js + Vite (أداء فائق وحصانة ضد أخطاء الـ Syntax).
* **التنسيق البصري**: Tailwind CSS + ستايل Glassmorphic داكن متوهج بنقاوة سيبرانية.
* **الباك إند والسيرفر**: Node.js + Express.js.
* **قاعدة البيانات والهوية**: Supabase السحابي المطور حياً.
* **بوابة المدفوعات**: بوابة دفع ميسر (Moyasar) السعودية المؤمنة بالكامل بنسبة 100%.
