# ⚡ TrendAura Core Backend Engine - V2 Enterprise Edition

High-performance, secure Node.js/Express server infrastructure engineered for real-time video script analytics and behavioral AI content generation.

## ⚙️ Deployment & Environment Architecture
- **Runtime Environment:** Node.js v18+ (ECMAScript Modules Execution)
- **Deployment Platform:** Hosted live via Railway Engine Framework using NIXPACKS Builder.
- **Sovereign Database:** Supabase Administrative Federation (Bypassing RLS securely via service roles).
- **AI Core Matrix:** Multi-model pipeline querying OpenRouter APIs with automated prompt caching tokens.

## 🛠️ Global Network Middleware Chains
1. **CORS Policy Handler:** Multi-origin bounds configured to securely absorb incoming traffic from Vercel layouts.
2. **DDOS Mitigation Guard:** Strict in-memory sliding scale tracking client IPs capped at 60 requests per minute.
3. **Federated Auth Validation:** Decoupled JWT fetch matrix dynamically authenticating user sessions directly with Supabase Edge servers.
4. **Centralized Error Hub:** Monitored catch-all framework isolating exceptions with hidden production stack traces.

## 🚀 Execution & Hot-Reload Commands
- Production Startup: `npm start`
- Development Environment: `npm run dev` (Powered by Nodemon file watch utilities)
